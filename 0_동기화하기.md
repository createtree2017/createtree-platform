# 환경 동기화 작업 지시서

> **AI 에이전트에게**: 이 파일을 읽고 아래 작업을 순서대로 실행해주세요.

---

## 작업 목표

이 프로젝트에서 bkit(이전에 사용하던 Gemini CLI 전용 플러그인) 잔여물을 모두 삭제하고, 새로운 PDCA 워크플로우 시스템만 남긴다.

## 실행할 작업

### 1. 프로젝트 안의 bkit 폴더 삭제

프로젝트 루트에 `~\.gemini\extensions\bkit\` 폴더가 있으면 `~` 폴더째로 삭제한다.

```powershell
Remove-Item -Recurse -Force "~\.gemini" -ErrorAction SilentlyContinue
```

### 2. 글로벌 GEMINI.md 삭제

`C:\Users\TOP\.gemini\GEMINI.md` 파일이 있으면 삭제한다. 프로젝트 루트에 이미 GEMINI.md가 있으므로 글로벌 파일은 불필요하다.

```powershell
Remove-Item "C:\Users\TOP\.gemini\GEMINI.md" -ErrorAction SilentlyContinue
```

### 3. Antigravity 내부 bkit 캐시 삭제

`C:\Users\TOP\.gemini\antigravity\code_tracker\active\` 안에 `bkit`이 이름에 포함된 폴더/파일이 있으면 모두 삭제한다.

```powershell
Get-ChildItem "C:\Users\TOP\.gemini\antigravity\code_tracker\active" -Filter "*bkit*" -ErrorAction SilentlyContinue | Remove-Item -Recurse -Force
```

### 4. 검증

아래 3가지를 확인한다:

1. 프로젝트 내에 `*bkit*` 이름의 파일/폴더가 없는지 확인 (node_modules 제외)
2. `C:\Users\TOP\.gemini\GEMINI.md`가 존재하지 않는지 확인
3. 프로젝트 루트에 `GEMINI.md`와 `.agent\workflows\` (6개 파일)가 존재하는지 확인

모두 정상이면 사용자에게 "동기화 완료"를 보고한다.

### 5. 이 파일 삭제

모든 작업이 완료되면 이 파일(`0_동기화하기.md`)을 삭제한다.
