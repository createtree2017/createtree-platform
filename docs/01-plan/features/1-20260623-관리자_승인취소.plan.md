# 관리자 승인 취소 Plan

## 배경

관리자 검수 화면에서 제출이 한 번 `approved`가 되면 기존 승인/보류 전용 API는 `submitted` 상태만 허용하므로 운영자가 승인 실수나 일정 변경을 되돌릴 수 없다. 승인 취소는 사용자 신청 취소가 아니므로 `cancelled`가 아니라 다시 검수 대기(`submitted`)로 돌리는 별도 운영 기능으로 구현한다.

## 목표

- 승인된 제출(`approved`)에만 관리자 승인 취소를 허용한다.
- 승인 취소 시 상태를 `submitted`로 되돌리고 `isLocked=false`, `reviewedBy=null`, `reviewedAt=null`로 초기화한다.
- 병원 관리자 권한 범위는 기존 검수 조회 정책과 동일하게 지킨다.
- 선착순 미션에서 승인 취소로 승인자가 빠지면 가장 빠른 대기자를 자동승격한다.
- 승인 취소 대상 사용자와 자동승격 사용자 모두 큰미션 진행률을 재계산한다.
- 검수 상세 모달에서 승인 상태일 때만 `승인 취소` 버튼을 표시한다.
