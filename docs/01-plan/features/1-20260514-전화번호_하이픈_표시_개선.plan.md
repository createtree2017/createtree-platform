# 전화번호 하이픈 표시 개선 Plan

## 작업 개요

회원가입과 고객/관리자 화면에서 전화번호를 숫자만 나열하지 않고 하이픈이 포함된 형태로 표시한다. 저장 데이터는 기존처럼 숫자만 유지한다.

## 목표

- 회원가입/프로필 완성/계정 설정 입력 중 `010-1234-5678` 형태로 표시한다.
- DB/API 전송 값은 `01012345678`처럼 숫자만 유지한다.
- 관리자 회원 목록, 회원 수정, 미션 제출 상세, 푸시 대상 선택 등 관리자 화면의 전화번호 표시를 하이픈 포맷으로 통일한다.
- 11자리는 `3-4-4`, 10자리는 `3-3-4`로 표시한다.

## 설계 메모

- 입력 필드는 하나로 유지한다.
- React Hook Form의 실제 필드 값은 숫자만 저장한다.
- `Input.value`에는 `formatPhoneNumber(field.value)`를 넣고, `onChange`에서는 `normalizePhoneNumberInput()`으로 숫자만 반영한다.
- 표시 UI는 `formatPhoneNumber(value) || '-'` 패턴을 사용한다.
