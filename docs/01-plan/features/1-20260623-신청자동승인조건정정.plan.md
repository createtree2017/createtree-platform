# 신청 자동승인 조건 정정 Plan

## 배경

문화센터 주제미션의 `신청` 세부미션은 현재 모집인원이 `0` 또는 비어 있을 때 `requireReview=false` 흐름을 타면서 선착순 여부와 관계없이 자동승인된다. 운영 정책은 자동승인을 `선착순(isFirstCome=true)`에만 허용해야 하므로 신청 제출 로직과 관리자 안내문구를 함께 정정한다.

## 목표

- `신청` 타입 세부미션은 `isFirstCome=true`일 때만 자동승인한다.
- `isFirstCome=false`이면 모집인원이 `0`이든 `N`이든 관리자 승인 대기(`submitted`)로 등록한다.
- `isFirstCome=true`이고 모집인원이 `0` 또는 비어 있으면 무제한 자동승인한다.
- `isFirstCome=true`이고 모집인원이 `N`이면 기존처럼 `N`명까지 자동승인하고 이후 대기자(`waitlist`)로 등록한다.
- 관리자 수동 승인 시 모집인원 초과 승인 가능 정책은 유지한다.
