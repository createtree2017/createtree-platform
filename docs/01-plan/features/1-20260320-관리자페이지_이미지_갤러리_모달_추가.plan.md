# 관리자페이지 이미지 갤러리 1K 원본 보기 모달 계획서 (Plan)

**작성일**: 2026-03-20
**작업 요약**: 갤러리 리스트 클릭 시 1024x1024 해상도의 원본 이미지를 팝업 모달로 확인하는 기능 추가

---

## 1. 목적 (Goal)
- 관리자가 생성된 이미지들의 저화질 리스트뿐만 아니라, 선택 시 1K 원본(1024x1024) 크기의 디테일한 이미지를 바로 확인할 수 있도록 편의성 제공

## 2. 변경 파일
- `client/src/components/admin/ImageGalleryAdmin.tsx`

## 3. 작업 절차 (Workflow)
1. `shadcn/ui`의 `Dialog` 컴포넌트 활용을 위해 import 추가
2. 기존에 정의되었으나 렌더링되지 않았던 `viewImageDialog`, `selectedImage` 상태 변수를 활용해 리스트 하단에 `Dialog` 컴포넌트 렌더링 구현
3. 모달 내에 다운로드, 공유, 상세 메타데이터를 포함한 UI 구성
