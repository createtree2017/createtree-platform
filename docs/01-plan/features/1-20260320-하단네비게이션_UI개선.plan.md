# 하단 네비게이션 UI 개선 계획서 (Plan)

## 1. 개요
* **작업 목표**: 모바일 하단 네비게이션을 5개 메뉴 구조로 재편하고, 최적화된 UX/UI(중앙 FAB + 은은한 테두리 반사광) 트렌드를 적용하여 고급스럽고 직관적인 인터페이스 제공
* **선정 디자인 (1번 타협안)**: 중앙 핵심 기능('AI 생성') 버튼 돌출형(FAB) + 좌우 평면 메뉴의 상단 1px 반사광 라인(유리 질감) + 중앙 버튼에 createTree 로고 삽입

## 3. 핵심 구현 내용

### 1) 상단 1px 은은한 반사광 라인 (Glassmorphism Edge)
* **적용 대상**: `BottomNavigation`의 최상단 부모 `nav` 컨테이너
* **스타일링 방안**:
  * 기존 `bg-background/80 backdrop-blur-xl` 유지하여 고급스러운 유리 질감 보존
  * 기존 투박했던 `border-t`를 제거하고, `box-shadow` 또는 `border-t border-white/10` 수준의 매우 투명하고 연한 라인 적용
  * 양쪽 모서리는 기기에 맞게 살짝 둥글게(예: `rounded-t-3xl`) 처리하여 부드러운 시작점 제공

### 2) 중앙 돌출형 버튼 (Floating Action Button - FAB)
* **적용 대상**: 메뉴 중 `menuId === 'ai-create'` (AI 생성) 메뉴
* **구현 방식**:
  * `absolute` 포지션을 활용하여 기존 네비게이션 라인(Top Border) 위쪽으로 튀어나오도록(`-top-6`) 위치 조정
  * 크고 둥근 원형태(`w-14 h-14 rounded-full`)
  * 배경색은 브랜드 컬러(보라/에메랄드) 혹은 쨍한 네온 그라데이션 적용 + 빛이 퍼지는 듯한 은은한 글로우 샤도우(`shadow-[0_0_15px_rgba(...)]`)
  * 아이콘 대신 앱의 **로고 이미지**(`/icons/icon-32x32.png` 등) 삽입하여 강력한 브랜딩 효과

### 3) 좌우 메뉴 평면화 & 비활성 상태 강조
* **적용 대상**: 중앙을 제외한 4개 메뉴 ('나의미션', '문화센터', '갤러리', 'MY')
* **구현 방식**:
  * 기존의 스케일 확장(`scale-110`) 및 그라데이션 박스 배경 효과 제거 (완벽한 평면 조연 역할)
  * 비활성화(Inactive) 상태: 아이콘을 얇은 선(Outline) 느낌 또는 투명도를 준 색상(`text-muted-foreground`)으로 처리
  * 활성화(Active) 상태: 선명한 흰색(또는 단색 브랜드 컬러) 면(Fill) 느낌 적용 및 하단 작은 점(Dot) 인디케이터 유지

---

## 4. 사이드 이펙트(영향도) 체크 및 방안
* **하단 여백**: 중앙 버튼이 위로 돌출되므로, 실제 페이지 렌더링 시 콘텐츠 최하단이 해당 버튼에 가려지지 않도록 `app` 메인 레이아웃의 `padding-bottom` 픽셀을 여유롭게 수정 필요 확인.
* **메뉴 개수**: 현재 `useMainMenus`에서 `나의미션`이 fallback_menus에 포함되었는지, 5개로 정상 출력되는지 재검토.
