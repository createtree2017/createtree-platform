## 1. 작업 배경 및 목표

## 2. 주요 설계 내용 (2단계)
1. **환경 매핑**
   - `Dev`: `develop` 브랜치와 연결. 중간 테스트 서버.
   - `Prod`: `main` 브랜치와 연결. 실제 서비스 서버.
2. **워크플로우**
   - `develop` 브랜치 작업 -> 자동 개발 배포 -> 검증.
   - `develop`을 `main`으로 머지 -> 자동 정식 배포.
