## 1. missions.tsx - 리스트 하단 거래처 중복 제거

### 변경 배경
문화센터 리스트 카드에 거래처(병원명)が 두 군데 표시되고 있었음:
- 제목 위 (배지 라인): `이즈맘산부인과 | 진행 중 | 선물` ← **유지**
- 하단 메타 정보: `🏥 이즈맘산부인과` ← **중복, 삭제**

### 변경 내용 (L325~332 제거)
```tsx
// 삭제된 코드 (Meta info 섹션 내부)
{mission.hospital && (
  <div className="flex items-center gap-2">
    <Building2 className="h-4 w-4" />
    <span>{mission.hospital.name}</span>
  </div>
)}
```

## 2. mission-detail.tsx - 세부페이지 제목 위 거래처 추가

### 변경 배경
미션 세부페이지 헤더에는 배지(진행중, 선물)만 표시되고 거래처명이 없었음.
리스트와 일관성 있는 UX를 위해 `거래처명 | 진행중 | 선물` 형태로 통일.

### 변경 내용 (L676~684 수정)
```tsx
// 기존
<MissionBadges ... className="mb-3" />

// 변경 후
<div className="flex items-center gap-1.5 flex-wrap mb-3">
  {mission.hospital && (
    <>
      <span className="text-xs font-medium text-muted-foreground">
        {mission.hospital.name}
      </span>
      <span className="text-muted-foreground/50 text-xs">|</span>
    </>
  )}
  <MissionBadges ... />
</div>
```
