# [작업명] 안드로이드 앱 아이콘 및 이름(AI문화센터) 수정

작업 일시: 2026.03.23

## 변경 파일 목록
- `assets/icon.png` (올바른 로고 이미지로 교체)
- `android/app/src/main/res/values/strings.xml` (앱 이름 변경: createtree -> AI문화센터)
- `android/app/src/main/res/...` (74개 해상도별 아이콘 및 스플래시 이미지 재생성)

## 핵심 변경 내용 및 히스토리
1. **앱 아이콘 이미지 정상화**
   - 기존 `assets/icon.png`에 에러 화면 스냅샷이 잘못 저장되어 있던 것을 확인 후, 사용자가 제공한 1024x1024 창조트리 로고 원본으로 교체 완료.
   - `npx capacitor-assets generate --android` 명령을 재실행하여 네이티브 폴더 내의 모든 아이콘 세트를 갱신함.
2. **앱 표시 이름(Launcher Label) 변경**
   - 기기 설치 시 아이콘 아래 표시되는 이름을 `createtree`에서 `AI문화센터`로 변경함.
   - `android/app/src/main/res/values/strings.xml`의 `app_name` 및 `title_activity_main` 스트링 수정.

## 동작 확인 상태 (✅/❌)
- ✅ `assets/icon.png` 파일 무결성 확인 (에러 화면 -> 정식 로고)
- ✅ 해상도별 아이콘 파일(74개) 자동 생성 완료
- ✅ `strings.xml` 내 한글 앱 이름 적용 확인

## 다음 작업 참고사항
- **중요**: 이 변경사항을 반영하려면 Android Studio에서 `versionCode 5` 이상으로 올려서 새 AAB 빌드 후 Play Console에 업로드해야 합니다.
- 구글 플레이 콘솔의 **'기본 스토어 등록정보'**에 등록된 512x512 아이콘도 웹에서 직접 교체해주셔야 스토어 화면에서 올바르게 노출됩니다.
- iOS 환경은 현재 `ios/` 폴더가 없으므로, 추후 Mac 환경 또는 클라우드 빌드 서비스를 통해 `npx cap add ios`부터 시작해야 합니다.
