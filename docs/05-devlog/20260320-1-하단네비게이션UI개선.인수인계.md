## 핵심 변경 내용

### 1. 중앙 FAB 돌출 버튼 (AI 생성 / 로고)
- `menuId === 'ai-create'` 메뉴를 네비게이션 라인 위로 30px 돌출되는 원형 FAB 버튼으로 분리 렌더링
- createTree 앱 로고(`/icons/icon-96x96.png`) 삽입
- 브랜드 그라데이션 배경 (purple → indigo → cyan) + 네온 글로우 박스 샤도우 적용
- 활성/비활성 상태별 글로우 강도 차등

### 2. 좌우 메뉴 평면화 (조연 역할)
- 기존의 보라색 배경 사각형 박스(`bg-gradient-to-br`, `scale-110`) 효과 완전히 제거
- **비활성 상태**: 얇은 아이콘(strokeWidth 1.5) + 진한 회색(`text-zinc-500`)으로 배경으로 물러남
- **활성 상태**: 두꺼운 아이콘(strokeWidth 2.5) + 흰색 + 약한 drop-shadow
- 아이콘 위 상단에 짧은(20px) 보라-인디고 그라데이션 라인으로 현재 위치 표시

### 3. 글래스모피즘 反사광 엣지 (상단 1px 라인)
- 기존 불투명한 테두리 제거
- `border-top: 1px solid rgba(255,255,255,0.07)` — 매우 투명한 흰색 반사광 라인
- `background: rgba(14,14,20,0.88)` + `backdrop-blur(20px)` 프리미엄 글래스모피즘 배경
- `box-shadow: 0 -4px 30px rgba(0,0,0,0.3), inset 0 1px 0 rgba(255,255,255,0.05)` 내부 반사광 효과

### 4. 나의미션 숨김 시 레이아웃 대응 (4개 ↔ 5개 메뉴)
- `ai-create`를 항상 중앙 FAB로 고정. 나머지 sideItems를 `floor` 기준으로 좌우 분배
- 좌측/우측 그룹을 각각 `flex-1 justify-around` 래퍼로 묶어 **메뉴 수와 무관하게 각 구역 내에서 자동 균형**
- 5개: `[나의미션, 문화센터] | FAB | [갤러리, MY]` — 각 구역 2개 균등 배치
- 4개(숨김): `[문화센터 중앙] | FAB | [갤러리, MY 균등]` — 왼쪽 1개는 구역 중앙, 오른쪽 2개는 구역 균등
- ai-create 메뉴가 API에서 없어질 경우의 fallback 처리(`<div>` 빈 공간 확보 포함)

## 20260320 추가 수정 이력

### A. FAB 위로 돌출 수정
- `absolute + top:-30` 방식 → `marginTop: -30` + `overflow: visible` 방식으로 변경
- 부모 `div`와 `nav`에 `overflow: visible` 명시하여 잘림 방지

### B. 네비게이션 바 디자인 개선
- 1단 테두리를 FAB에서 네비바 전체로 이동: `border: 1px solid rgba(139,92,246,0.35)`
- 상단 양끝 라운드 처리: `borderRadius: '20px 20px 0 0'`
- FAB 자체에서 링 테두리 제거, 글로우 빛만 유지

### C. 스크롤바 우측 이격 현상 해결
- `App.tsx`의 `<main>` `padding-bottom` → `margin-bottom`으로 변경
- 스크롤바가 네비게이션 시작점 바로 위에서 끝나도록 수정

### D. 좌우 메뉴 균등 정렬 (방법 2 적용)
- 좌측/우측 그룹을 `flex-1 + justify-around` 래퍼로 묶음
- 메뉴 수 변동 시에도 AI생성이 항상 정중앙 고정, 각 구역 내에서 자동 균형
