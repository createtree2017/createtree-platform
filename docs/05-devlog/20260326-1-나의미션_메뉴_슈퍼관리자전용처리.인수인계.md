# 나의미션 메뉴 슈퍼관리자 전용 처리

## 1. 개요
* **날짜:** 2026-03-26
* **작업자:** AI Agent
* **목적:** 하단 네비게이션(Bottom Navigation)에 위치한 "나의미션" 메뉴를 슈퍼관리자(superadmin) 계정에서만 보이도록 접근 권한을 제한함. 이후 서비스 정식 오픈 시 일반 사용자에게도 공개될 것을 고려해, 손쉽게 설정을 변경할 수 있도록 프론트엔드 레벨에서 처리.

## 2. 변경 파일 목록
* `client/src/hooks/useMainMenus.ts`

## 3. 핵심 변경 내용 
* **프론트엔드 필터링 로직 추가:** 
  * `useAuth()` 훅을 사용하여 현재 활성화된 세션의 사용자 정보를 가져옵니다.
  * `user?.memberType === 'superadmin'` 조건을 확인하여 슈퍼관리자 여부를 판별합니다.
  * 하단 네비게이션 아이템을 구성하는 `navItems` 배열에서, `menuId === 'culture-center'` (나의미션)인 항목은 슈퍼관리자일 경우에만 필터를 통과하도록 구현했습니다.
  * 일반 로그인 유저 및 비로그인 상태의 유저에게는 해당 탭이 UI 상에서 렌더링되지 않습니다.

## 4. 리뷰 및 동작 확인 상태
* ✅ `useAuth` 시스템 연동 및 토큰 기반 슈퍼관리자 확인 완료
* ✅ `filter()` 메서드를 통해 원본 데이터를 해치지 않으면서 UI 렌더링만 제한

## 5. 주의사항 및 알려진 이슈
* **서비스 오픈 시의 변경점:** 추후 정식 오픈시 '나의미션' 접근 제한을 해제하려면 `client/src/hooks/useMainMenus.ts` 내의 해당 `filter` 구문(`item.menuId === 'culture-center'` 조건)을 제거하거나 주석 처리하면 즉시 모든 사용자에게 오픈됩니다.
* 이 처리는 "UI 요소"를 숨기는 것으로, 메뉴 버튼 자체를 가려 접속 흐름을 제한하는 의도입니다. 만약 URL (`/missions`) 경로의 직접 접근까지 완벽히 차단해야 한다면 라우팅 컴포넌트(`App.tsx` 등) 또는 해당 페이지(`client/src/pages/missions.tsx` 등)에 `Redirect`/`Unauthorized` 접근 제어 코드가 추가로 필요합니다.
