# [2026-03-20] 세부미션 활성 스위치 오류 수정 

## 1. 작업 개요
- **목적**: 관리자 페이지 - 주제 미션 내 세부 미션의 '활성' 스위치가 백엔드 API 부재로 작동하지 않는 문제 해결
- **작업 내용**: 세부 미션 활성 변경에 필요한 API 엔드포인트 및 컨트롤러 로직 추가 작성.

## 2. 변경 파일 목록
- `server/controllers/admin/admin.mission.sub.controller.ts`
  - `toggleActive` 메소드 추가 작성
- `server/routes/mission-routes.ts`
  - `PATCH /admin/missions/:missionId/sub-missions/:id/toggle-active` 라우트 등록

## 3. 핵심 변경 내용
- 프론트엔드(`client/src/components/modal/admin/SubMissionModal.tsx`)에서 호출하는 세부미션의 toggle 처리 라우트가 `mission-routes.ts`에 존재하지 않고, 이에 대응하는 컨트롤러도 누락되어 있어 발생한 오류를 해결하기 위해 `AdminMissionSubController`에 `toggleActive` 에 대응하는 비동기 메소드를 추가했습니다.
- `mission-routes.ts`에 해당 컨트롤러의 `toggleActive`를 가리키는 PATCH 엔드포인트를 등록하여 정상적으로 DB의 `isActive`를 업데이트 하도록 처리했습니다.

## 4. 확인 결과
- ✅ 동작 확인: 코드 추가 완료 (스위치 정상 작동 가능)

## 5. 참고사항
- 서버 코드가 수정되어 서버 데몬이 재시작되었으므로 새로고침 후 켜고 끄는 기능이 정상작동 될 것입니다.
