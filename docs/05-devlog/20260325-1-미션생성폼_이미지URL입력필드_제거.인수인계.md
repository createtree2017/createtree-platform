## 변경 배경 및 의사결정 이유

- 해당 Input은 관리자가 외부 이미지 URL을 **직접 붙여넣기** 위한 보조 수단이었으나, 실제로 외부 URL을 직접 입력하는 사용 시나리오가 없음
- 이미지 업로드 버튼으로 GCS에 업로드하면, 반환된 URL이 `form.setValue()`로 폼 값에 자동 저장되므로 UI상 입력 필드 없이도 기능 동일
- UI상 이미지 URL이 노출되면서 공간을 차지하고 혼란을 줄 수 있어 관리자가 제거 요청

---

## 동작 확인 상태

- ✅ 이미지 업로드 버튼 → 파일 선택 → GCS 업로드 → URL 자동 저장: 정상 동작
- ✅ 미리보기 이미지 표시 및 X 버튼(삭제) 정상 동작
- ✅ 저장 시 URL 값 서버 전송 정상
