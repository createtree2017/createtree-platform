## 변경 배경

프로젝트 내에 아래 3개(+1 래퍼)의 중복 토스트 구현체가 혼재하여 동작이 일관되지 않음:

| 파일 | REMOVE_DELAY | 자동닫힘 | 실제 사용 |
|------|-------------|---------|---------|
| `hooks/use-toast.ts` | 1,000,000ms | ❌ | ✅ 전체 프로젝트 |
| `components/ui/use-toast.ts` | 1,000ms + 3초 dismiss | ✅ | ❌ (래퍼 경유만) |
| `components/ui/use-toast.tsx` | 1,000,000ms | ❌ | ❌ 미사용 |
| `hooks/useToast.ts` | - | - | 일부 파일 경유 |

**핵심 문제**: `Toaster.tsx`가 `hooks/use-toast.ts`를 사용하는데 이 파일에는 자동닫힘이 없어서 토스트가 영구적으로 화면에 남아 있었음.

```
모든 파일

      ↓
  DEFAULT_DURATION = 3000ms (기본 3초 후 자동닫힘)
  toast({ title: "...", duration: 5000 })  // duration으로 개별 조정 가능
```
